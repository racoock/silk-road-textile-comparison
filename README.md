# Silk Road Textile Comparison Database

A collaborative, structured database of key textile artifacts from China and Israel along the Silk Road. This database serves as a primary resource for comparative analysis, focusing on evidence of technological exchange, trade in materials, and shared motifs.

## Project Structure

- `/china/` - Artifact entries from China
- `/israel/` - Artifact entries from Israel
- `artifact_template.md` - Template for new artifact entries

## How to Contribute

1. Review the `artifact_template.md` for the required fields.
2. Open an Issue to propose a new artifact entry, using appropriate labels (e.g., `china`, `silk`, `proposed-entry`).
3. Create a new branch for your entry.
4. Add a markdown file following the template in the appropriate regional folder.
5. Submit a Pull Request for peer review.
6. After addressing feedback, merge the PR to add the artifact to the database.

## Artifact Entry Template

Each artifact entry should follow the structure defined in `artifact_template.md`. Please copy the template and fill out all fields.
